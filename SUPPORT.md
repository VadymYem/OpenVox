# Support

## Bugs

Use the repository bug-report template and include:

- browser and exact version;
- operating system and device class;
- whether the site is installed as a PWA;
- audio input device when the issue is audio-related;
- reproduction steps;
- console errors;
- whether the issue reproduces after disabling browser audio enhancements.

## Feature requests

Use the feature-request template. Explain the real workflow the feature should improve rather than only proposing an interface control.

## Security

Do not report unpatched vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).
