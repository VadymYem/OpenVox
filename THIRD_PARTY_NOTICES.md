# Third-Party Notices

OpenVox uses open-source packages installed through npm. The authoritative license text for every dependency is distributed with its package and should be reviewed together with `package-lock.json` for each release.

## Direct runtime dependencies

- React — MIT
- React DOM — MIT
- React Router DOM — MIT
- `@tonejs/midi` — MIT

## Direct development and quality dependencies

The development toolchain includes TypeScript, Vite, the Vite React plugin, ESLint, Prettier, Vitest, Testing Library, axe-core, jsdom and related type/test utilities under their respective open-source licenses.

OpenVox does not bundle Workbox or `vite-plugin-pwa`; the production manifest and service worker are generated by the repository's own post-build script.

Before a public release, review the complete locked dependency tree and the license metadata contained in the installed packages.
